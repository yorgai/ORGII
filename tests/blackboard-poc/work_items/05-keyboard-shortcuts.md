# Add keyboard shortcuts overlay

## Requirements

- Pressing "?" opens a keyboard shortcuts modal/overlay
- The overlay lists all available keyboard shortcuts grouped by category (navigation, editing, general)
- ESC key or clicking outside the modal closes it
- No backend changes needed — this is purely a frontend UI feature
- The modal should have a proper focus trap for accessibility (tab stays within modal)
- Styled consistently with the existing application theme

## Acceptance Criteria

1. Pressing "?" key opens the shortcuts overlay
2. ESC key or clicking the backdrop closes the overlay
3. Shortcuts are organized into clear categories with labels
4. Focus is trapped within the modal while it is open
5. The modal is styled consistently and looks good in both light and dark themes
