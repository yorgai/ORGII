## 2026 年 6 月

### Agent Org 群聊与协调

- Agent Org 团队对话现在默认显示群聊视图，支持实时气泡流、进度指示器和非中断消息路由，不会打断成员正在进行的工作。
- 新增成员直接消息、@提及路由和基于轮次的对话协调，使多智能体对话更加清晰。
- 新增干预状态界面，让团队工作流中的人工监督时刻一目了然。

### WorkStation 与源代码管理

- 重新设计 Source Control 侧边栏中的 PR 区域，采用共享 token、紧凑操作，并在 PR 关闭或合并后提供更安全的重新创建行为。
- 在 WorkStation 中新增 GitHub Issues 浏览功能，包含侧边栏连接和与 Source Control 对齐的筛选器。
- 新增差异选择操作，可复制文件引用并将所选更改发送到聊天，加快代码审查提示速度。
- 在聊天编辑器中新增 Push-to-Talk 语音输入。

### Canvas

- Canvas 产物现在可以从智能体输出流入聊天和 WorkStation，以内联块或专用预览标签页的形式呈现。
- 新增 Canvas 模拟器应用界面、`setup_repo` 聊天块和聊天跳转至模拟器功能，提供更丰富的生成应用预览。

### 会话与回放

- 新增会话 JSON 导入/导出，便于移动或检查会话数据。
- 新增 Benchmark Runner 执行模式和可复用的批量任务管理流程。
- 对 SQLite 写入器进行串行化处理，防止高并发活动时出现会话数据库锁争用。

## 2026 年 5 月

### Agent Org 运行时

- Agent Org 成员现在作为真正可执行的会话运行，包含启动快照、唤醒语义、原子任务认领和成员状态证据，以实现更丰富的运行时追踪。
- 新增暂停和恢复控制（替代停止），状态在应用重启后保持，重新启动时恢复侧边栏会话。
- 改进了 Worktree 感知子智能体，使成员运行能够使用准确的仓库上下文。
- 新增任务队列证据并锁定证据分类规则，确保运行时分类的一致性。

### 身份验证与提供商

- 新增 Gemini、Claude Code 和 Codex 的 OAuth 流程，涵盖嵌入式登录、token 刷新持久化和账号切换隔离。
- 新增 GitHub 设备流身份验证，内置公共客户端 ID 并支持本地 token 回退。
- 改进 Cursor CLI 和 API 密钥兼容性，减少提供商配置的边缘情况。

### WorkStation

- 新增全标签页停靠导航和跨主机标签页切换。
- 引入存在感胶囊，将用户可用性上下文注入当前活跃的智能体会话。
- 新增原生 Linear 项目视图，可在 ORGII 内查看与 Linear 关联的项目数据。
- 新增计划模式审批流程，提供更清晰的智能体计划审查状态。
- 在聊天中新增实时流式传输指标：已用时间、Token 速率和完成时间估算。

### 智能体配置

- 在智能体配置页面新增共享运行时限制区域，采用更严格的能力默认值和默认关闭的工具初始化。
- 将规则、记忆和进化整合到统一的设置界面。
- 改进 SOUL.md 处理，确保个性化上下文一致地应用于预期的智能体流程。
